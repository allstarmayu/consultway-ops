# Palette Reference

## Files

- `warm-ambient-palette.pdf` — warm consulting aesthetic (cream, walnut, terracotta)

## Which palette does this portal use?

**This internal operations portal uses navy + red** — see `app/globals.css` for the live CSS variables and `docs/07-design-system.md` for the spec. That matches the Figma wireframes in `docs/design/figma-screenshots/`.

## Why is the warm ambient palette in this repo?

It is **reference material for a related project**: the public-facing Consultway website at `consultway.info`. That site targets prospective clients and deserves a softer, consulting-firm aesthetic. The warm ambient palette is appropriate there.

Since the two projects share brand ownership, we keep this palette here for easy reference. It is **not** applied to the portal.
